# Sử Dụng Git & GitHub Cho Dự Án

## 🚀 1. Khởi tạo dự án lần đầu tiên

```bash
git init
git add .
git commit -m "feat: Initial commit with standardized Arduino structure and clean VL53L0X filter"
git branch -M main
git remote add origin https://github.com/MyDreamIsHere0511/BallAndBeam-2026.git
git push -u origin main
git push -u origin main --force
```
---

## 🔥 2. Đẩy Code Lên Sau Mỗi Lần
### Bước 1: Lưu toàn bộ thay đổi
```bash
git add .
```
### Bước 2: Đặt tên cho bản lưu 
```bash
git commit -m "comment"
```
### Bước 3: Đẩy thẳng lên máy chủ GitHub
```bash
git push
```
